# 💰 Finance Dashboard

A modern and responsive finance dashboard built with Next.js, React, and Tailwind CSS.

## 🚀 Live Demo

👉 https://new-finance-dashboard.vercel.app/

---

## ✨ Features

- 📊 Interactive dashboard with charts
- ➕ Add transactions (income & expenses)
- ✏️ Edit transactions
- 🗑️ Delete transactions
- 💾 Data persistence using localStorage
- 🌗 Dark / Light mode
- 📱 Fully responsive design

---

## 🧠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Recharts

---

## ⚙️ Getting Started

Clone the project:

```bash
git clone https://github.com/neto112/finance-dashboard.git
```

Install dependencies:

`npm install`

Run the project:

`npm run dev`
