# Finance Dashboard

A modern finance dashboard built with Next.js, React, and Tailwind CSS.

## Features

- Add / Edit / Delete transactions
- Dark / Light mode
- Responsive layout
- Data persistence with localStorage
- Interactive charts

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Recharts
